# To JSON

## This is a Python script that converts any text file to JSON.

Example:
```
This is a example
this is a new line
these are some numbers and punctuation: 1, 12, 123, 2, 3, 4, 5, 6
```

Output:
```
{
  "numbers": [
    "1",
    "12",
    "123",
    "2",
    "3",
    "4",
    "5",
    "6"
  ],
  "a": [
    "a",
    "a",
    "and",
    "are"
  ],
  "e": [
    "example"
  ],
  "i": [
    "is",
    "is"
  ],
  "l": [
    "line"
  ],
  "n": [
    "new",
    "numbers"
  ],
  "p": [
    "punctuation"
  ],
  "s": [
    "some"
  ],
  "t": [
    "these",
    "this",
    "this"
  ]
}
```
